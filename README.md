# Analisis-de-datos-en-SQL-para-app-de-lectura-de-libros-
Este proyecto tiene como objetivo analizar la base de datos de una aplicación de lectura para extraer información clave sobre libros, autores, editoriales y el comportamiento de los usuarios. El propósito final es generar hallazgos que sirvan como insumo para diseñar la propuesta de valor de un nuevo producto digital de lectura.

## Objetivos del proyecto

- Procesar y explorar la base de datos de la app de lectura.
- Identificar patrones en libros, autores y editoriales.
- Analizar el comportamiento de los usuarios (calificaciones y reseñas).
- Extraer insights para una propuesta de valor del nuevo producto.
- Proporcionar recomendaciones basadas en datos para estrategias editoriales y de comunidad.
